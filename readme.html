🎯 Giới thiệu

Barber Spa là nền tảng đặt lịch salon/barber trực tuyến trên nền tảng WordPress, kết nối khách hàng với các salon chuyên nghiệp. Hệ thống được tích hợp dưới dạng Plugin **LOPAS** mạnh mẽ, hỗ trợ quản lý đa chi nhánh và quy trình đặt lịch tự động.

Khách hàng

Tìm kiếm, đặt lịch & thanh toán trực tuyến.

Chủ Salon

Quản lý dịch vụ, nhân viên & lịch hẹn cơ sở.

Quản trị viên

Điều hành toàn hệ thống & phê duyệt salon.

🚀 Công nghệ sử dụng

Core & Backend
WordPress 6.x PHP 8.0 (Plugin MVC) MySQL / MariaDB
Frontend & UI
HTML5/CSS3 Bootstrap 5 JavaScript (AJAX) Chart.js
Integrations
VNPay Sandbox PHPMailer (SMTP)

🗂️ Cấu trúc thư mục trọng tâm

Dự án tập trung vào Plugin lopas trong thư mục plugins của WordPress:

wordpress/
├── wp-content/
│   └── plugins/
│       └── lopas/             <-- Plugin chính
│           ├── assets/        (CSS, JS, Images)
│           ├── includes/      (Core logic, Admin, API)
│           ├── templates/     (Giao diện đặt lịch)
│           └── lopas.php      (Khởi tạo plugin)
├── wp-config.php              (Cấu hình Database)
└── README.md                  (Tài liệu gốc)

✨ Chức năng chính

  • Tìm kiếm salon theo khu vực/dịch vụ (Autocomplete).
  • Đặt lịch 4 bước: Dịch vụ -> Thợ -> Giờ -> Xác nhận.
  • Giữ slot 10 phút để tránh đặt trùng.
  • Thanh toán VNPay hoặc tại quầy.
  • Quản lý My Bookings & Đánh giá dịch vụ.

  • Dashboard thống kê doanh thu/booking (Chart.js).
  • Quản lý chi tiết Salon, Dịch vụ & Nhân viên.
  • Phê duyệt/Từ chối/Hoàn thành lịch hẹn.
  • Quản lý đánh giá và phản hồi khách hàng.

⚙️ Hướng dẫn cài đặt

Yêu cầu: XAMPP, PHP 8.0+, WordPress.
  1. Copy Source Code
    Vào thư mục htdocs/Wordpressnangcao/wordpress.
  2. Cấu hình Database
    Tạo DB wordpress_db và import file SQL dự án.
  3. Kích hoạt Plugin
    Vào WP-Admin -> Plugins -> Activate "LOPAS Barber Spa".
